## Overview

Portfolio OS is not a demo—it is a deployed product that powers lead generation, content distribution, and automation. This page aggregates the economics behind the platform so hiring managers and clients can see tangible ROI.

---

## Cost Profile

| Category | Monthly Cost | Notes |
| -------- | ------------ | ----- |
| Vercel hosting | $20 | Site, Docs, Dashboard on the Pro plan |
| OpenAI usage | $5–$20 | Depends on chatbot conversations (streaming via GPT-4o mini) |
| Google APIs | &lt;$1 | Calendar + Meet usage well within free tier |
| Email (Resend) | $0–$10 | Pay-as-you-go based on contact form volume |

**Total monthly run cost:** typically &lt;$35 while offering capabilities comparable to SaaS bundles that exceed $200/month.

---

## Value Drivers

1. **Automated lead capture** – Chatbot answers technical questions instantly and routes high-intent visitors to booking.
2. **Self-serve scheduling** – Booking System eliminates context switching and ensures I meet prospects in their timezone.
3. **Operational readiness** – Built-in dashboard provides analytics, CMS, and deployment scripts so features ship quickly.
4. **Credibility** – Demonstrates ability to design, implement, document, and operate a production platform solo.
