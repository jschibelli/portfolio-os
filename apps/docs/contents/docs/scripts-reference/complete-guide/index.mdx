#### `automate-pr-unified.ps1`

<Note type="tip" title="Complete PR Automation">
This is the primary PR automation script that orchestrates AI-powered analysis, quality checks, and automated responses.
</Note>

Complete PR automation with AI-powered analysis.

<Tabs defaultValue="basic">
  <TabsList>
    <TabsTrigger value="basic">Basic Usage</TabsTrigger>
    <TabsTrigger value="advanced">Advanced</TabsTrigger>
    <TabsTrigger value="actions">Actions</TabsTrigger>
  </TabsList>

  <TabsContent value="basic">
```powershell
# Full PR automation
.\automate-pr-unified.ps1 -PRNumber 150 -Action all

# Analyze only
.\automate-pr-unified.ps1 -PRNumber 150 -Action analyze

# Generate responses
.\automate-pr-unified.ps1 -PRNumber 150 -Action respond
```
  </TabsContent>

  <TabsContent value="advanced">
```powershell
# With auto-fix enabled
.\automate-pr-unified.ps1 -PRNumber 150 -Action all -AutoFix

# Continuous monitoring
.\automate-pr-unified.ps1 -PRNumber 150 -Action monitor -Interval 300

```
  </TabsContent>

  <TabsContent value="actions">
<div>

**Available Actions:**
- `analyze` - AI-powered PR analysis
- `respond` - Generate automated responses
- `monitor` - Continuous PR monitoring
- `all` - Complete workflow (analyze + respond + monitor)

**Parameters:**
- `-PRNumber` - Pull request number (required)
- `-Action` - Action to perform (required)
- `-AutoFix` - Enable automatic fixes
- `-Interval` - Monitoring interval in seconds
- `-AIModel` - Specify AI model for analysis

</div>

Core system management

- `destroy` - Remove worktrees

- `assign` - Assign issues
- `sync` - Synchronize
